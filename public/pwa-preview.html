Cómo Funcionan los Splash Screens

  • Android: Se genera automáticamente usando el icono 512px + color de fondo blanco + nombre "Movi Digital"
  • iOS: Se usan las imágenes splash personalizadas para cada modelo de iPhone
  • Navegadores PWA: Muestran el icono 512px centrado mientras carga la aplicación
  • Duración: Aparecen solo 1-2 segundos mientras la app se inicializa
