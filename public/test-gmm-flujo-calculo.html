⚠️ REGLA DE ORO:
Si la prima base final no es correcta, ninguna prima adicional será correcta.
Por eso: NUNCA se debe "ajustar" la prima adicional, siempre se debe corregir primero la prima base.

📋 Orden Correcto del Cálculo (NO ALTERAR)

Paso 1️⃣: Calcular Prima Base Final
Prima Base Final = Base Edad/Sexo × Factor Estado × Factor Nivel × Factor Tabulador × Factor SA × Factor Deducible × Factor Coaseguro ÷ (1 − Σ Cargas)
Validaciones:
✓ Todos los factores deben ser > 0
✓ Suma de cargas debe ser < 1
✓ Resultado debe redondearse a 2 decimales
Paso 2️⃣: Para cada cobertura seleccionada
Prima Cobertura = Prima Base Final × Coeficiente Cobertura
Validaciones:
✓ Solo coberturas con coverage_enabled = true
✓ Coeficiente debe estar definido y ser numérico
✓ Si coeficiente = 0 → prima cobertura = 0 (válido)
✓ Resultado debe redondearse a 2 decimales
Paso 3️⃣: Sumar todas las primas de coberturas
Prima Adicional Asegurado = Σ Prima Cobertura_i
Paso 4️⃣: Obtener Prima Total Asegurado
Prima Total = Prima Base Final + Prima Adicional

🧪 Ejemplo de Cálculo Correcto

Caso de Prueba:

Entrada:
Prima Base Final = $18,604.00
Coberturas seleccionadas: Medicamentos fuera (0.204711) + VIP (0.0344)

Cálculo:
Prima Medicamentos = $18,604.00 × 0.204711 = $3,807.66
Prima VIP = $18,604.00 × 0.0344 = $639.58
Prima Adicional Total = $3,807.66 + $639.58 = $4,447.24

Resultado Final:
Prima Base Final: $18,604.00
Prima Adicional: $4,447.24
Prima Total: $23,051.24

🔒 Validaciones Obligatorias (Anti-Errores)

Validaciones de Datos
  • ✓ prima_base_final > 0
  • ✓ coverage_coef es numérico y definido
  • ✓ coverage_enabled = true
Validaciones de Consistencia
  • ✓ Si una cobertura existe en UI pero no en Excel → ERROR
  • ✓ Si el Excel cambia coeficiente → recalcular todo
  • ✓ Si coverage_coef = 0 → prima cobertura = 0 (válido)
Validaciones de Orden
  • ✓ Bloquear cálculo si cargas no aplicadas
  • ✓ Bloquear cálculo si prima base no finalizada
📌 Nota Importante sobre Terminología:

En el código actual, hay dos variables con nombres confusos:
• primaBaseFinal = prima ANTES de aplicar cargas
• primaBaseConCargas = prima DESPUÉS de aplicar cargas = lo que el Excel llama "Prima Base Final"

Las coberturas se deben calcular sobre primaBaseConCargas (que es la verdadera "Prima Base Final" según el Excel).
✅ IMPLEMENTACIÓN ACTUAL:
El código YA está usando primaBaseConCargas como base para calcular coberturas adicionales, lo cual es CORRECTO según las especificaciones del Excel.

Solo necesitamos validar que los COEFICIENTES sean exactos.
