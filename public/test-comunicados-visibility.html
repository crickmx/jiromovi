🧪 Test de Visibilidad de Comunicados

📋 Objetivo del Test

Verificar que las políticas RLS muestren correctamente los comunicados según las reglas de visibilidad:

  • Caso 1: Comunicado sin reglas de visibilidad → Visible para todos
  • Caso 2: Comunicado con para_todos = true → Visible para todos
  • Caso 3: Comunicado con reglas específicas de rol → Visible solo para usuarios con ese rol
  • Caso 4: Comunicado con reglas específicas de oficina → Visible solo para usuarios de esa oficina
